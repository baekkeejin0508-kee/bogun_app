# 보건간호 총정리 통합 앱 - 언제 어디서나 간편하게 복습하는 모바일 학습 메이트

서울의료보건고등학교 학생들의 국가고시 및 학업 성취도 향상을 지원하는 맞춤형 웹 서비스, '보건간호 총정리 통합 앱'을 제출합니다.

## 1. 기획 의도
보건간호 과목은 외워야 할 개념과 법규가 방대하여 학생들이 학습에 어려움을 겪는 경우가 많습니다. 특히 흩어져 있는 자료들을 매번 찾아보고 외우는 것은 비효율적이며 지루함을 유발합니다.

그래서 저는 "아이들이 늘 손에 쥐고 있는 스마트폰으로 언제 어디서든 게임처럼 쉽게 복습할 수 있다면 어떨까?"라는 아이디어에서 출발했습니다. 무거운 책 대신 직관적인 인터페이스를 통해 학생들이 필수 개념을 확인하고 퀴즈로 스스로 실력을 점검할 수 있는 '반복 학습 최적화 앱'을 구상하게 되었습니다.

## 2. 핵심 기능
### 1) 직관적인 플래시카드 (개념 암기)
학생들은 각 단원별(인구와 출산, 모자보건, 지역사회간호, 의료관계법규, 감염병 등)로 세분화된 플래시카드를 통해 핵심 질문과 정답을 터치 한 번으로 쉽게 확인할 수 있습니다. 시각적인 카드 뒤집기 애니메이션을 적용하여 암기의 지루함을 덜고 학습에 대한 몰입도를 높입니다.

### 2) 실전 퀴즈 및 즉각적인 피드백
플래시카드로 학습한 내용을 바탕으로 '실전 퀴즈'를 풉니다. 문제를 풀 때마다 정답과 오답에 대한 즉각적인 해설(Feedback) 팝업을 제공하여 틀린 부분을 완벽히 이해하고 넘어갈 수 있도록 돕습니다. 퀴즈가 끝나면 최종 정답률과 맞힌 문제 수를 시각적으로 보여주어 학생 스스로 성취도를 직관적으로 파악하게 합니다.

## 3. 강력한 안전 및 프라이버시 보호
학생들이 앱을 시작하기 전, 안전한 디지털 학습 환경을 위해 '중등윤리 핵심 가이드', '개인정보처리방침', '에듀집 필수기준 체크리스트', '학교용 필수 기준 체크리스트' 4가지 항목의 문서를 필수로 열람하고 동의해야만 로그인 화면으로 진입할 수 있도록 시스템을 설계했습니다.

또한 수집하는 개인정보를 학습 식별을 위한 최소한의 '학번'과 '이름'으로 제한하였으며, 학년도 종료 시 해당 데이터를 즉각 파기함을 방침에 명시하여 학생들의 프라이버시를 강력히 보호합니다.

## 4. 언제 어디서든 접속 가능한 접근성 (오프라인/로컬 구동 최적화)
현재 이 앱은 별도의 무거운 백엔드 서버(Firebase 등) 없이 프론트엔드 환경에서 가볍고 빠르게 구동되도록 최적화되어 있습니다. 네트워크 환경이 불안정한 교실이나 이동 중에도 끊김 없이 쾌적하게 학습할 수 있습니다. Vercel을 통해 배포되어 특정 기기에 구애받지 않고 스마트폰, 태블릿, PC의 웹 브라우저만 있으면 언제든 즉시 접속이 가능합니다.

## 5. 기술 스택 (Tech Stack)
- **Frontend:** HTML5, Tailwind CSS (모바일 반응형 UI 설계), Vanilla JavaScript
- **Backend & Database:** 서버리스 기반 로컬 구동 최적화 (별도 DB 연결 없이 경량화)
- **Deployment:** Vercel (배포 주소: https://bogun-app.vercel.app/)
- **Repository:** GitHub (https://github.com/baekkeejin0508-kee/bogun_app)

## 6. 기대 효과
본 '보건간호 총정리 통합 앱'은 교사의 학습 자료 준비 및 오프라인 유인물 인쇄 부담을 획기적으로 줄여주는 동시에, 학생들에게는 자투리 시간을 활용할 수 있는 자기주도적 학습 환경을 제공합니다. 기술을 통해 '모든 학생이 포기하지 않고 보건간호의 핵심을 쉽고 재미있게 마스터하는 교실'을 만들어가는 것이 이 프로젝트의 최종 목표입니다.
