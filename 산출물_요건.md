# 교사 개발자 성장형 산출물 조건 확인

본 보건앱은 '교사 개발자 성장형 산출물 조건'에 따라 다음과 같이 구성되었습니다.

## 01. 소속학교·이름 표기
- **위치**: 앱 최초 진입 화면(필수 확인 및 로그인 화면) 하단에 명확하게 표기
- **표기 내용**: "소속학교: 서울의료보건고등학교 / 이름: 백 기진"

## 02. 개인정보처리방침
- **내용**: 회원가입(로그인) 시 수집 항목(학번, 이름), 이용 목적(학습 이력 관리), 보관 기간(해당 학년도 종료 시 파기), 문의처 등이 기재된 표준 개인정보처리방침을 포함
- **연결**: 동의 화면 및 앱 하단 푸터에서 개인정보처리방침 파일(md) 및 모달로 연결

## 03. 사용약관
- **내용**: 서비스 이용 조건, 책임 범위, 무료 교육용 서비스로서의 책임 제한, 금지 행위(정보 도용 등), 변경 안내 포함
- **연결**: 앱 하단 푸터에서 이용약관 팝업으로 명확하게 연결

## 04. GitHub 레포지토리 주소
- **주소**: 소스코드 공개 주소 제공 (예: https://github.com/baekkeejin0508-kee/bogun_app)
- 개발 이력 및 커밋 내용을 통해 앱의 변경 사항 및 협업 내역 확인 가능

## 05. Firebase 백엔드
- 본 앱은 현재 Firebase 연동 없이 로컬 데이터 기반으로 동작하고 있으나, 향후 학생별 퀴즈 정답률, 로그인 정보 및 학습 데이터를 실시간으로 저장하고 인증하기 위해 Firebase 백엔드 연동을 고려하여 설계됨. (추후 고도화 시 반영 가능)

## 06. 프론트엔드 웹주소
- **주소**: Vercel 또는 GitHub Pages 등을 통해 배포 완료 후 발급되는 실제 접속 가능한 URL
- 로컬이 아닌 외부에서도 누구나 접속 가능하도록 배포

## 07. 사용방법 안내
- **로그인**: 앱 시작 전 필수 확인 4종(윤리, 개인정보, 에듀집, 학교용 체크리스트) 동의 후, '학번'과 '이름'만으로 간편하게 로그인
- **주요 기능**: 원하는 단원 선택, 플래시카드(개념 학습) 및 실전 퀴즈(성취도 확인) 기능
- **주의사항**: 학습용으로만 사용하며 다른 목적으로의 이용 금지 안내

## 08. 배포 완료 확인
- **웹 접속**: 배포된 주소로 정상 접속됨
- **링크 작동**: 약관, 방침 및 필수 체크리스트 파일들이 정상적으로 열림
- **정책 페이지 연결**: 동의 화면과 푸터의 약관/방침 모달 정상 작동
- **저장 기능 정상 여부**: 퀴즈 결과 점수 정상 표시 및 상태 변경 확인
